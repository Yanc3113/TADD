Video Classification Benchmark

To perform benchmarks for video classification, first clone the following repository: mmaction2. Navigate to the directory configs/recognition/swin or configs/recognition/uniformerv2, and select the appropriate configuration file based on your model variant (e.g., base or large). Adjust the parameters within the selected YAML configuration file accordingly.
Critical parameters requiring adjustment include data_root paths. If you're evaluating our pretrained checkpoints, update the pretrained field in the official YAML file with the corresponding checkpoint URL provided by our repository.
Different evaluation metrics correspond to different configuration files. The configuration files for video classification benchmarks are stored within this project's repository. Please access them through the provided links.
We highly encourage developers to create new benchmarks by customizing our existing configuration files as needed.

Few-shot Learning Benchmark

To conduct benchmarks for few-shot classification tasks, begin by cloning the following repositories: MoLo and Knowledge Prompting for FSAR.
After cloning, ensure critical parameters such as data_root are accurately configured. For evaluating our pretrained checkpoints, update the pretrained parameter within the YAML configuration files. Note that different evaluation metrics necessitate distinct configuration setups.
Refer to our repository for detailed configuration file examples, accessible via: Few-Shot Configuration Files.
Navigate to the main directory of MoLo with the command:
cd /root/MoLo-master
Then execute the benchmarking script by running:
python runs/run.py --cfg configs/projects/MoLo/ucf101/3way_1shot_car_rgb.yaml
